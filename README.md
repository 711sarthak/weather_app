
# Weather App (Flutter)

## Getting Started

### Prerequisites
**Flutter**

### Installing

**API Key**

Open a file called `api_keys.dart` in `lib/src/api/`

Edit ('YOUR_API_KEY' with API key you generated from openweathermap, Get it here: https://openweathermap.org/):
  ```
  class ApiKey {
    static const openWeatherMap = "YOUR_API_KEY";
  }
  ```
