## Getting Started

### Prerequisites
**Flutter**

### Installing

**API Key**

GET 'YOUR_API_KEY' from openweathermap, Get it here: https://openweathermap.org/)


Create a file called `api_keys.dart` in `lib/src/api/`

Add this in `api_keys.dart`:
  ```
  class ApiKey {
    static const openWeatherMap = "YOUR_API_KEY";
  }
  ```
